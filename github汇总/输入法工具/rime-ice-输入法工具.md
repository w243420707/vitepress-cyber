# 项目名称

# 作者或组织名
iDvel

# 项目类别
输入法工具

# 核心痛点
Rime输入法词库不够完善，缺乏长期维护的高质量简体中文词库。

# 主要功能
长期维护的简体中文词库，为Rime输入法提供极佳支持，包含雾凇拼音配置，总星数超过13000。

# 项目亮点
- 长期维护的简体中文词库
- 为Rime输入法提供极佳支持
- 包含雾凇拼音配置
- 总星数超过13000
- 支持通过东风破工具一键安装
