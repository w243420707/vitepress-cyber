# 项目名称
Gleam

# 作者或组织名
gleam-lang

# 项目类别
编程语言

# 核心痛点
传统编程语言在构建高并发系统时，缺乏类型安全保障，容易在运行时出现错误；代码维护困难，并发处理复杂。

# 主要功能
一种类型安全的函数式编程语言，用于构建可扩展的并发系统。

# 项目亮点
1. 类型安全，编译时即可发现大量错误
2. 函数式编程特性，代码简洁易维护
3. 并发处理优雅，适合构建高并发系统
4. 可编译到Erlang虚拟机或JavaScript，跨平台支持
5. 学习曲线友好，适合函数式编程新手和老手
