# 💻 编程语言

## 概述

编程语言是软件开发的基石，不同的编程语言适用于不同的场景和需求。本分类汇集了各类优质的编程语言，包括现代系统编程语言、函数式编程语言、AI专用语言等，帮助您选择合适的语言进行开发。

## 语言列表

### Gleam

- **项目名称**: Gleam
- **作者**: gleam-lang
- **核心功能**: 类型安全的函数式编程语言，用于构建可扩展的并发系统，可编译到 Erlang 虚拟机或 JavaScript
- **主要亮点**:
  - 类型安全，编译时即可发现大量错误
  - 函数式编程特性，代码简洁易维护
  - 并发处理优雅，适合构建高并发系统
  - 可编译到 Erlang 虚拟机或 JavaScript，跨平台支持
  - 学习曲线友好，适合函数式编程新手和老手
- **详细介绍**: [Gleam 完整文档](/github汇总/编程语言/Gleam-编程语言)
- **GitHub 地址**: https://github.com/gleam-lang/gleam

---

### Go

- **项目名称**: Go
- **作者**: The Go Authors
- **核心功能**: 简洁易读的语法，超快的编译速度，内置强大的并发编程支持，跨平台编译，标准库丰富
- **主要亮点**:
  - 由谷歌主导开发，数千名社区开发者共同贡献
  - 完全开源免费，无 vendor lock-in
  - 语法简洁，代码可读性高
  - 编译速度极快，开发体验流畅
  - 内置并发模型，轻松构建高性能应用
  - 云原生应用开发的主流选择
- **详细介绍**: [Go 完整文档](/github汇总/编程语言/Go-编程语言)
- **GitHub 地址**: https://github.com/golang/go

---

### inkos

- **项目名称**: inkos
- **作者**: Narcooo
- **核心功能**: 受 Rust 和 Lua 影响的小语言，与 Lua 运行时兼容，可编译为 Lua 源代码
- **主要亮点**:
  - 提供比 Lua 更安全、更强大的语言选择
  - 保持 Lua 的灵活性和简洁性
  - 为喜欢 Rust 安全特性但需要 Lua 脚本便利性的开发者提供折中方案
  - 可以利用现有的 Lua 生态系统
- **详细介绍**: [inkos 完整文档](/github汇总/编程语言/inkos-编程语言)
- **GitHub 地址**: https://github.com/Narcooo/inkos

---

### Mojo

- **项目名称**: Mojo
- **作者**: Modular
- **核心功能**: 统一的 AI 开发和部署平台，MAX 框架，Mojo 编程语言（Python 的超集），性能堪比 C/C++，专为 AI 优化
- **主要亮点**:
  - Modular 官方项目，质量可靠
  - 同时提供框架和编程语言
  - Mojo 是 Python 的超集，学习成本低
  - 性能堪比 C/C++
  - 专为 AI 优化
  - 完全开源，免费使用
- **详细介绍**: [Mojo 完整文档](/github汇总/编程语言/Mojo-编程语言)
- **GitHub 地址**: https://github.com/modularml/mojo

---

### Zig

- **项目名称**: Zig
- **作者**: Zig Software Foundation
- **核心功能**: 通用的编程语言和工具链，语法简洁，编译速度快，简单的交叉编译，可直接调用 C 代码，强大的编译期计算能力
- **主要亮点**:
  - 现代的系统编程语言
  - 语法简洁，没有隐藏控制流、隐藏内存分配、预处理器、宏
  - 编译速度特别快，零依赖
  - 简单的交叉编译
  - 可直接调用 C 代码，不需要绑定，也能被 C 代码调用
  - 强大的编译期计算能力，可在编译期运行代码优化性能
  - 可选的内存管理，可选择手动管理或使用标准库分配器
  - 完善的工具链，内置构建系统、打包器、测试框架等
  - 目前最受关注的现代系统编程语言之一
- **详细介绍**: [Zig 完整文档](/github汇总/编程语言/Zig-编程语言)
- **GitHub 地址**: https://github.com/ziglang/zig

## 分类特点

本分类编程语言涵盖以下类型：

- **系统编程语言** - Go、Zig
- **函数式编程语言** - Gleam
- **AI 专用语言** - Mojo
- **脚本语言变体** - inkos

## 语言选择建议

1. **云原生开发** - 推荐使用 Go
2. **系统编程** - 推荐使用 Zig
3. **高并发系统** - 推荐使用 Gleam
4. **AI 开发** - 推荐使用 Mojo
5. **Lua 生态集成** - 推荐使用 inkos
