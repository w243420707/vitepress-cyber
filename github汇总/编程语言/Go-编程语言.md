# 安装或部署教程

## 教程适用对象与环境范围
适用于想要学习和使用Go语言进行开发的开发者，支持Windows、macOS、Linux等主流操作系统。

## 前置条件检查
1. 系统要求：Windows 10及以上、macOS 10.15及以上、主流Linux发行版
2. 磁盘空间：至少500MB可用空间
3. 网络要求：能够访问官方下载站点（如无法访问，参考 [海外网络环境配置 →](/网络环境)）

## 主路径：二进制安装（推荐新手）

### 步骤1：下载Go
**步骤目标**：获取Go官方二进制安装包
**执行位置**：浏览器
**完整命令或操作**：访问 https://go.dev/dl/ 下载适合你操作系统的安装包
**成功标志**：下载完成
**失败处理**：检查网络连接，或参考 [海外网络环境配置 →](/网络环境)

### 步骤2：安装Go
**步骤目标**：完成Go的安装
**执行位置**：本机
**完整命令或操作**：
- Windows：运行下载的.msi安装程序，按提示完成安装
- macOS：运行下载的.pkg安装程序，按提示完成安装
- Linux：解压下载的.tar.gz到/usr/local目录
**成功标志**：安装程序提示安装完成
**失败处理**：查看安装日志，排查权限问题

### 步骤3：验证安装
**步骤目标**：确认Go安装成功
**执行位置**：命令行终端
**完整命令或操作**：运行 `go version`
**成功标志**：显示Go版本号，如 `go version go1.21.0 windows/amd64`
**失败处理**：检查环境变量PATH是否包含Go的bin目录

## 安装成功验证
在命令行运行 `go version`，应显示Go版本信息；运行 `go env` 可查看Go的环境配置。

## 常见错误排查
1. **现象**：命令行找不到go命令
   - **可能原因**：环境变量PATH未配置
   - **解决步骤**：将Go的bin目录（默认Windows是C:\Program Files\Go\bin，macOS/Linux是/usr/local/go/bin）添加到系统PATH环境变量中
   
2. **现象**：下载速度慢或无法下载
   - **可能原因**：网络问题
   - **解决步骤**：参考 [海外网络环境配置 →](/网络环境)，或使用国内镜像站点

3. **现象**：编译程序时报错
   - **可能原因**：代码问题或Go版本不兼容
   - **解决步骤**：检查代码语法，确认Go版本满足项目要求

## 备选部署路径：源码安装
如二进制安装包不适用你的系统，可参考 https://go.dev/doc/install/source 进行源码安装（如无法访问，参考 [海外网络环境配置 →](/网络环境)）。
