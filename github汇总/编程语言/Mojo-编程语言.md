# 安装或部署教程

## 教程适用对象与环境范围
适用于想要使用Mojo进行AI开发的开发者，支持macOS和Linux。

## 前置条件检查
1. 系统要求：macOS 14或更高、Ubuntu 22.04/24.04、Debian 12、Fedora 40/41
2. 网络要求：能够访问Modular官方下载站点
3. Python版本：Python 3.10-3.13（如使用MAX Python）

## 主路径：快速安装（推荐新手）

### 步骤1：安装Mojo CLI
**步骤目标**：在系统上安装Mojo命令行工具
**执行位置**：命令行终端
**完整命令或操作**：
- macOS/Linux：运行 `curl -s https://get.modular.com | sh`
**成功标志**：安装完成提示
**失败处理**：检查网络连接，或参考 [海外网络环境配置 →](/网络环境)

### 步骤2：验证安装
**步骤目标**：确认Mojo安装成功
**执行位置**：命令行终端
**完整命令或操作**：运行 `mojo --version`
**成功标志**：显示Mojo版本号
**失败处理**：检查环境变量PATH是否配置

### 步骤3：运行第一个程序
**步骤目标**：创建并运行一个简单的Mojo程序
**执行位置**：命令行终端和代码编辑器
**完整命令或操作**：
1. 创建hello.mojo文件，写入 `fn main(): print("Hello, world!")`
2. 运行 `mojo hello.mojo`
**成功标志**：输出Hello, world!
**失败处理**：检查代码语法

## 安装成功验证
运行 `mojo --version` 显示版本信息，运行示例程序成功。

## 常见错误排查
1. **现象**：命令行找不到mojo命令
   - **可能原因**：环境变量PATH未配置
   - **解决步骤**：将Mojo的bin目录添加到系统PATH环境变量

2. **现象**：网络安装失败
   - **可能原因**：网络问题
   - **解决步骤**：参考 [海外网络环境配置 →](/网络环境)

3. **现象**：代码运行错误
   - **可能原因**：Mojo语法问题
   - **解决步骤**：查看官方文档学习Mojo语法

## 备选部署路径：Docker
如本地安装不适用，可使用Docker镜像。
