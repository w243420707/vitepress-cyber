# 🔐 安全工具

## 概述

安全工具是保障系统安全、进行安全审计和渗透测试的重要工具。本分类汇集了各类优质的开源安全工具，涵盖网络安全、漏洞扫描、渗透测试等多个领域，帮助您提升系统安全性。

## 工具列表

### NetBird

- **项目名称**: NetBird
- **作者**: netbirdio
- **核心功能**: 基于 WireGuard 的无配置点对点专用网络工具，无需打开端口、配置防火墙规则，支持细粒度访问控制
- **主要亮点**: 
  - 无配置，使用简单
  - 基于 WireGuard，安全高效
  - 细粒度访问控制
  - 企业级特性（SSO、MFA、审计日志等）
  - 跨平台支持，几乎所有平台都支持
  - 有云服务可以直接使用，也支持自托管
- **详细介绍**: [NetBird 完整文档](/github汇总/安全工具/netbird-安全工具)
- **GitHub 地址**: https://github.com/netbirdio/netbird

---

### pentagi

- **项目名称**: pentagi
- **作者**: vxcontrol
- **核心功能**: AI 驱动的自动化渗透测试工具，在隔离的 Docker 沙箱环境中自主执行安全测试，内置 20 多种专业安全工具
- **主要亮点**:
  - 完全自主的 AI 驱动渗透测试
  - Docker 沙箱隔离，安全可控
  - 内置 20 多种专业安全工具
  - 智能记忆系统和知识图谱
  - 支持多种 LLM 提供商
  - 可完全本地化部署
  - 提供完整的 REST 和 GraphQL API
- **详细介绍**: [pentagi 完整文档](/github汇总/安全工具/pentagi-安全工具)
- **GitHub 地址**: https://github.com/vxcontrol/pentagi

---

### Trivy

- **项目名称**: Trivy
- **作者**: aquasecurity
- **核心功能**: 全面多功能的安全扫描器，可扫描容器镜像、文件系统、Git 仓库、虚拟机镜像、Kubernetes，发现漏洞、配置问题、敏感信息泄露、软件许可证问题
- **主要亮点**:
  - 一个工具完成多种安全扫描
  - 支持容器镜像、文件系统、Git 仓库、Kubernetes 等多种目标
  - 可发现漏洞、配置问题、敏感信息、许可证问题
  - 集成众多主流平台和工具
  - Aqua Security 开源项目
- **详细介绍**: [Trivy 完整文档](/github汇总/安全工具/trivy-安全工具)
- **GitHub 地址**: https://github.com/aquasecurity/trivy

## 分类特点

本分类安全工具涵盖以下领域：

- **网络安全** - VPN、网络访问控制等
- **渗透测试** - 自动化安全测试工具
- **漏洞扫描** - 容器、代码、基础设施扫描
- **安全审计** - 配置审计、敏感信息检测

## 使用建议

1. **根据需求选择** - 根据您的具体安全需求选择合适的工具
2. **充分测试** - 在生产环境使用前，先在测试环境充分验证
3. **持续更新** - 安全工具更新频繁，建议保持最新版本
4. **合规使用** - 请在合法合规的前提下使用安全工具
