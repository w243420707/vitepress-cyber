# 项目名称
Ventoy

# 作者或组织名
ventoy

# 项目类别
系统工具

# 核心痛点
创建可启动USB驱动器需要反复格式化，使用多个系统镜像不方便

# 主要功能
- 创建可启动USB驱动器
- 支持直接从ISO/WIM/IMG/VHD(x)/EFI文件启动
- 一次复制多个镜像文件，启动时选择
- 支持x86 Legacy BIOS、IA32 UEFI、x86_64 UEFI、ARM64 UEFI等
- 支持MBR和GPT分区风格

# 项目亮点
- 不需要反复格式化磁盘
- 支持大多数类型的操作系统
- 已测试1300多个ISO文件
- 功能丰富，支持持久化、自动安装等
