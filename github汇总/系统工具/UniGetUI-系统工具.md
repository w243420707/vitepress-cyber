# 项目名称
UniGetUI

# 作者或组织名
marticliment

# 项目类别
系统工具

# 核心痛点
Windows包管理分散，缺乏一个统一的图形界面包管理器。

# 主要功能
适用于Windows的统一包管理器图形界面，支持多种包管理后端，前身为WingetUI，允许用户轻松搜索、安装和管理软件包，GitHub Star数已超过26000。

# 项目亮点
- 适用于Windows的统一包管理器图形界面
- 支持多种包管理后端
- 前身为WingetUI
- 允许用户轻松搜索、安装和管理软件包
- GitHub Star数已超过26000
- 界面友好，操作简单
