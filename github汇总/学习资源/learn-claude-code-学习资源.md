# 作者或组织名

# 项目类别
学习资源

# 核心痛点
想理解AI编程Agent的工作原理但缺乏简单易懂的教程；复杂的框架和算法让初学者望而却步。

# 主要功能
从0到1构建迷你Claude Code风格的AI编程助手，用Bash实现，让你理解AI编程Agent的工作原理。

# 项目亮点
1. 从0到1构建迷你Claude Code风格AI编程助手
2. 用Bash实现，不用复杂框架，不用高深算法
3. 核心理念：模型本身就是Agent
4. GitHub标星数增长迅速，超过两万星
5. 适合学习AI编程Agent的工作原理
