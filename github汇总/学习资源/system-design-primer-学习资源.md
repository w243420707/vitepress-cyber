# system-design-primer

## 作者或组织名
Donne Martin

## 项目类别
学习资源

## 核心痛点
想学习系统设计或准备相关面试，经常找不到系统全面的资料

## 主要功能
- 学习如何设计大规模系统
- 准备系统设计面试
- 系统设计基础教程（面向对象设计、架构设计等）
- 系统设计关键主题（可扩展性、可用性、一致性、缓存、负载均衡等）
- 大量真实系统设计面试题及详细解答
- 社区驱动，大量贡献者参与维护
- 支持多种语言

## 项目亮点
- 系统全面的系统设计学习资源
- 从基础教程到关键主题全覆盖
- 大量真实系统设计面试题及详细解答（设计URL缩短服务、设计Twitter、设计YouTube等）
- 社区驱动，大量贡献者参与维护
- 支持多种语言
- GitHub上超高Stars，最受欢迎的系统设计学习资源之一
