# developer-roadmap

## 作者或组织名
Kamran Ahmed

## 项目类别
学习资源

## 核心痛点
想学习编程或某个技术方向，经常不知道从哪开始、学什么

## 主要功能
- 社区驱动的开发者路线图、指南和其他教育内容
- 各种技术方向的路线图（前端、后端、DevOps、全栈、AI、数据科学、区块链等）
- 互动式路线图，可点击查看详情、标记进度、分享路线图
- 大量的最佳实践、指南、文章资源
- 社区贡献和维护
- 开源项目，任何人都可以贡献内容

## 项目亮点
- 覆盖各种技术方向的路线图
- 互动式路线图，可查看详情、标记进度、分享
- 大量的最佳实践、指南、文章资源
- 社区驱动，内容由社区贡献和维护
- 开源项目，任何人都可以贡献
- 网站界面清晰易读，查找方便
- GitHub上超高Stars，最受欢迎的开发者学习资源之一

## 安装或部署教程
这是一个学习资源网站项目，主要用于访问和学习，不需要部署。

### 使用方式
1. 访问官方网站：https://roadmap.sh [海外网络环境配置 →](/网络环境)
2. 选择感兴趣的技术方向路线图
3. 点击查看每个主题的详细内容
4. 可以标记自己的学习进度
5. 可以分享自己的路线图
