# 项目名称
Gopeed

# 作者或组织名
GopeedLab

# 项目类别
下载工具

# 核心痛点
下载工具不够好用，缺乏支持多种协议的跨平台全能下载管理器。

# 主要功能
使用Golang和Flutter开发的高速下载管理器，支持HTTP、BitTorrent、Magnet等多种协议，兼容所有主流平台，跨平台支持Windows、macOS、Linux、Android等。

# 项目亮点
- 使用Golang和Flutter开发
- 高速下载管理器
- 支持HTTP协议
- 支持BitTorrent协议
- 支持Magnet协议
- 兼容所有主流平台
- 跨平台支持Windows
- 跨平台支持macOS
- 跨平台支持Linux
- 跨平台支持Android
- 全称Go Speed，意为极速下载
- 是一款轻量却强大的现代下载管理器
