# 项目名称
BaiduPCS-Go

# 作者或组织名
qjfoidnh

# 项目类别
下载工具

# 核心痛点
百度网盘网页版和官方客户端不够灵活，缺乏命令行操作方式。

# 主要功能
基于Go语言开发的百度网盘命令行客户端，提供类似Linux shell的文件处理命令，支持Windows、macOS、Linux等多平台，集成分享链接和秒传链接转存功能。

# 项目亮点
- 基于Go语言开发
- 提供类似Linux shell的文件处理命令
- 支持Windows、macOS、Linux等多平台
- 可在移动设备上通过终端模拟器运行
- 集成分享链接和秒传链接转存功能

# 安装或部署教程

## 教程适用对象与环境范围
适用于需要通过命令行管理百度网盘的用户，支持Windows、macOS、Linux等系统。

## 前置条件检查
1. 系统版本：Windows、macOS、Linux等
2. Go语言环境（如需从源码编译）
3. 网络连接正常
4. 百度网盘账号

## 主路径：安装与启动

### Windows安装
1. **步骤目标：下载并安装Windows版本**
   - **执行位置：** 本机
   - **完整命令或操作：** 访问项目Releases页面下载最新版本的Windows二进制文件，解压缩并将其移动到%PATH%指定的任何目录
   - **成功标志：** 可在命令行中运行BaiduPCS-Go命令
   - **失败处理：** 检查文件是否正确解压，路径是否添加到环境变量

### 从源码编译安装
1. **步骤目标：克隆项目并编译**
   - **执行位置：** 项目目录
   - **完整命令或操作：**
     ```bash
     git clone https://github.com/qjfoidnh/BaiduPCS-Go.git
     cd BaiduPCS-Go
     go build
     ```
   - **成功标志：** 编译成功，生成可执行文件
   - **失败处理：** 检查Go语言环境是否正确安装，依赖是否完整

详细步骤请参考项目官方文档 [项目文档](https://github.com/qjfoidnh/BaiduPCS-Go) [海外网络环境配置 →](/网络环境)

## 安装成功验证
在命令行中运行`BaiduPCS-Go`命令，查看是否正常启动并显示帮助信息。

## 常见错误排查
1. **现象：命令无法识别**
   - **可能原因：** 可执行文件未添加到环境变量
   - **解决步骤：** 将可执行文件所在目录添加到系统PATH环境变量

2. **现象：编译失败**
   - **可能原因：** Go语言环境未正确安装或版本不兼容
   - **解决步骤：** 检查Go语言版本，确保满足项目要求

3. **现象：网络连接错误**
   - **可能原因：** 网络连接问题或API限制
   - **解决步骤：** 检查网络连接，查看项目Issue页面寻找解决方案
