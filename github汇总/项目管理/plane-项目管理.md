# 项目名称
plane

# 作者或组织名
makeplane

# 项目类别
项目管理

# 核心痛点
项目管理工具要么太笨重要么功能不全，缺乏一个现代化的开源项目管理平台。

# 主要功能
开源项目管理工具，功能丰富，类似Linear，支持工作空间与项目管理、周期规划、任务看板、需求管理、里程碑设定、文档与协作、时间追踪、自动化与集成，GitHub Star数已突破38000。
