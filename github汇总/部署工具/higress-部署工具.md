# 项目名称
higress

# 作者或组织名
阿里云（Alibaba）

# 项目类别
部署工具

# 核心痛点
云原生架构中API网关和微服务网关的构建复杂，缺乏统一的高性能解决方案。

# 主要功能
阿里云开源的云原生网关，基于Istio和Envoy构建，提供强大的API网关和微服务网关能力。

# 项目亮点
1. 由阿里云开源
2. 基于Istio和Envoy构建
3. 提供强大的API网关和微服务网关能力
4. 是云原生架构中的重要组件
5. 采用Apache-2.0许可证
6. 已获得超过1.4k星标
7. 是构建云原生应用架构的核心基础设施
