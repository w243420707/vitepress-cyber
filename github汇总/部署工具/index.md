# 🚀 部署工具

## 概述

部署工具是应用程序部署和运维的重要组成部分。本分类汇集了各类优质的开源部署工具，涵盖 PaaS 平台、API 网关、路由器固件、安装辅助工具等，帮助您简化部署流程，提升运维效率。

## 工具列表

### Coolify

- **项目名称**: Coolify
- **作者**: coolify-org
- **核心功能**: 开源自托管 PaaS 平台，支持多种应用类型部署，免费 SSL 证书自动配置，多种常用数据库一键部署，反向代理支持
- **主要亮点**:
  - 完全免费开源，MIT 协议
  - GitHub 星标超过 2.1 万
  - 可完美替代 Heroku 和 Netlify
  - 无需复杂配置，新手也能轻松使用
  - 自托管，完全掌控部署过程
- **详细介绍**: [Coolify 完整文档](/github汇总/部署工具/Coolify-部署工具)
- **GitHub 地址**: https://github.com/coolify-org/coolify

- **主要亮点**:
  - 由阿里云开源
  - 基于 Istio 和 Envoy 构建
  - 提供强大的 API 网关和微服务网关能力
  - 是云原生架构中的重要组件

  - 已获得超过 1.4k 星标
  - 是构建云原生应用架构的核心基础设施
- **详细介绍**: [higress 完整文档](/github汇总/部署工具/higress-部署工具)

## 分类特点

本分类部署工具涵盖以下类型：

- **PaaS 平台** - Coolify
- **API 网关** - higress
- **路由器固件** - lede
- **安装辅助工具** - OpenClawInstaller、openclaw-uninstaller

## 工具选择建议

1. **应用部署** - 推荐使用 Coolify
2. **云原生网关** - 推荐使用 higress
3. **路由器定制** - 推荐使用 lede
4. **OpenClaw 管理** - 推荐使用 OpenClawInstaller 和 openclaw-uninstaller
