# 项目名称
Coolify

# 作者或组织名
coolify-org

# 项目类别
部署工具

# 核心痛点
1. 依赖第三方云平台部署应用，缺乏自主权
2. 应用部署流程复杂，配置繁琐
3. SSL证书管理麻烦
4. 数据库部署和维护成本高
5. 新手难以快速上手部署

# 主要功能
1. 开源自托管PaaS平台
2. 支持多种应用类型部署
3. 免费SSL证书自动配置
4. 多种常用数据库一键部署
5. 项目一键安装与升级
6. 反向代理支持
7. 监控与状态仪表板
8. 备份与恢复策略
9. HTTP重定向与基本认证

# 项目亮点
1. 完全免费开源，MIT协议
2. GitHub星标超过2.1万
3. 可完美替代Heroku和Netlify
4. 无需复杂配置，新手也能轻松使用
5. 自托管，完全掌控部署过程
