# 项目名称
openclaw-uninstaller

# 作者或组织名
inclusionAI

# 项目类别
部署工具

# 核心痛点
- 卸载OpenClaw不干净，留下残留文件
- 配置文件残留影响重新安装
- 缺少官方的卸载工具
- 手动卸载复杂，容易出错

# 主要功能
- OpenClaw卸载工具
- 帮助用户干净、完整地卸载OpenClaw
- 配合OpenClawInstaller使用
- 不留任何残留文件或配置
- 让安装和卸载都变得简单放心

# 项目亮点
- 官方卸载工具，可靠性高
- 干净完整卸载，无残留
- 配合OpenClawInstaller使用
- 操作简单，使用放心
- 由inclusionAI社区开发
