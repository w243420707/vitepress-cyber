# pyrite64

## 项目名称
pyrite64

## 作者或组织名
HailToDodongo

## 项目类别
游戏开发工具

## 核心痛点
游戏开发爱好者想要为复古游戏机开发游戏但缺乏合适的工具。

## 主要功能
N64游戏引擎和可视化编辑器，使用Libdragon和tiny3d创建可在真实N64主机或精确模拟器上运行的3D游戏。支持从Blender导入3D模型（GLTF格式），支持fast64材质。支持HDR加泛光渲染、大纹理渲染（256x256）。运行时引擎处理场景管理、渲染、碰撞检测、音频等核心功能。提供节点图编辑器编写基础控制流程。全局资源管理器自动处理内存清理。Windows平台自动工具链安装。

## 项目亮点
- 不使用任何任天堂专有SDK或库
- 支持Blender模型导入和fast64材质
- HDR加泛光渲染
- 大纹理渲染（256x256）
- 节点图编辑器
- 全局资源管理器
- Windows自动工具链安装

## 安装或部署教程

### 教程适用对象与环境范围
本教程适用于想要使用Pyrite64开发N64游戏的开发者。

### 前置条件检查

1. **检查操作系统**
   - 执行命令：无
   - 通过标准：Windows系统（推荐），或Linux/macOS

2. **检查Blender是否安装**（用于3D模型导入）
   - 执行命令：`blender -v`
   - 通过标准：显示Blender版本号

### 主路径：Windows安装

**步骤1：克隆项目**
- 目标：获取项目源码
- 位置：本地目录
- 命令：`git clone https://github.com/HailToDodongo/pyrite64.git`
- 成功标志：项目目录创建成功
- 失败处理：检查网络连接 [海外网络环境配置 →](/网络环境)

**步骤2：阅读文档**
- 目标：了解构建步骤
- 位置：浏览器
- 操作：访问项目文档 https://github.com/HailToDodongo/pyrite64/tree/main/docs
- 成功标志：理解构建流程
- 失败处理：查看FAQ

**步骤3：构建编辑器**
- 目标：编译编辑器
- 位置：项目目录
- 操作：按照文档中的构建指南执行
- 成功标志：编辑器编译成功
- 失败处理：检查依赖是否完整

**步骤4：运行编辑器**
- 目标：启动编辑器
- 位置：项目目录
- 操作：运行编译好的编辑器可执行文件
- 成功标志：编辑器窗口打开
- 失败处理：检查系统兼容性

### 备选路径：使用模拟器测试

**步骤1：下载精确模拟器**
- 目标：获取Ares或gopher64模拟器
- 位置：浏览器
- 操作：访问对应模拟器官网下载
- 成功标志：模拟器下载完成
- 失败处理：检查网络连接

**步骤2：配置模拟器**
- 目标：设置模拟器环境
- 位置：模拟器设置
- 操作：按照模拟器文档配置
- 成功标志：模拟器可以运行N64 ROM
- 失败处理：查阅模拟器文档

### 安装成功验证
- 编辑器正常启动
- 可以创建新项目
- 可以导入Blender模型

### 常见错误排查

1. **工具链安装失败**
   - 可能原因：网络问题或权限问题
   - 解决步骤：以管理员权限运行，检查网络连接

2. **模型导入失败**
   - 可能原因：模型格式不支持
   - 解决步骤：确保使用GLTF格式，检查fast64材质设置

3. **模拟器无法运行游戏**
   - 可能原因：模拟器版本不兼容
   - 解决步骤：使用Ares v147或更新版本
