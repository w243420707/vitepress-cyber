# 项目名称
pua

# 作者或组织名
tanweai

# 项目类别
AI工具

# 核心痛点
AI助手工作效率不高，缺乏有效的提示词优化方案，AI工具使用体验有待提升。

# 主要功能
将大厂话术转化为AI System Prompt，适配Claude Code、Cursor等主流Agent，支持多语言（英语、日语等），提升AI工作效率。

# 项目亮点
- 将大厂话术转化为AI System Prompt
- 适配Claude Code、Cursor等主流Agent
- 支持多语言（英语、日语等）
- Star数量持续上涨
- 提升AI工作效率
