# voicebox

## 项目名称
voicebox

## 作者或组织名
jamiepine

## 项目类别
AI工具

## 核心痛点
想要使用AI语音但担心隐私泄露，不想付费订阅云服务。

## 主要功能
开源本地语音克隆工作室，完全本地运行保护隐私。支持四种TTS引擎（Qwen3-TTS、LuxTTS、Chatterbox多语言版、Chatterbox快速版），覆盖23种语言。支持用几秒钟音频克隆声音。提供8种后期处理效果（变调、混响、延迟、合唱/镶边、压缩、增益、高通滤波、低通滤波）。故事编辑器支持多声音时间轴编辑，创建对话、播客和叙事内容。支持无限长度生成（自动分块和交叉淡入淡出）。提供REST API用于集成到其他项目。

## 项目亮点
- 完全本地运行，保护隐私
- 支持4种TTS引擎，23种语言
- 8种后期处理效果
- 故事编辑器支持多声音时间轴
- 无限长度生成
- 提供REST API
- 支持macOS、Windows、Linux
- 原生性能（Tauri + Rust）

## 安装或部署教程

### 教程适用对象与环境范围
本教程适用于想要使用Voicebox进行语音克隆和生成的用户。

### 主路径：下载桌面应用（推荐）

**步骤1：访问下载页面**
- 目标：获取应用下载链接
- 位置：浏览器
- 操作：访问 https://voicebox.sh 或 GitHub Releases
- 成功标志：找到下载链接
- 失败处理：检查网络连接 [海外网络环境配置 →](/网络环境)

**步骤2：下载对应平台版本**
- 目标：下载安装包
- 位置：浏览器
- 操作：选择对应平台下载（macOS DMG、Windows MSI）
- 成功标志：安装包下载完成
- 失败处理：检查磁盘空间

**步骤3：安装应用**
- 目标：安装Voicebox
- 位置：本地
- 操作：运行安装包，按提示完成安装
- 成功标志：应用安装成功
- 失败处理：检查系统权限

**步骤4：启动应用**
- 目标：运行Voicebox
- 位置：桌面或应用菜单
- 操作：打开Voicebox应用
- 成功标志：应用窗口打开
- 失败处理：检查系统兼容性

### 备选路径：Docker部署

**步骤1：克隆项目**
- 目标：获取项目源码
- 位置：终端
- 命令：`git clone https://github.com/jamiepine/voicebox.git`
- 成功标志：项目克隆成功
- 失败处理：检查网络连接

**步骤2：启动Docker服务**
- 目标：使用Docker Compose启动
- 位置：项目目录
- 命令：`docker compose up`
- 成功标志：服务启动成功
- 失败处理：检查Docker是否运行

### 安装成功验证
- 应用正常启动
- 可以创建新的语音配置文件
- 可以输入文本并生成语音

### 常见错误排查

1. **macOS提示"无法验证开发者"**
   - 可能原因：应用未签名
   - 解决步骤：系统设置 → 隐私与安全 → 仍要打开

2. **GPU加速不工作**
   - 可能原因：缺少GPU驱动或CUDA
   - 解决步骤：安装对应平台的GPU驱动

3. **模型下载失败**
   - 可能原因：网络问题
   - 解决步骤：配置代理或手动下载模型
