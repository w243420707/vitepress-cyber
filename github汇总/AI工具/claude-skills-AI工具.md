# 项目名称
claude-skills

# 作者或组织名
alirezarezvani

# 项目类别
AI工具

# 核心痛点
- 使用Claude需要手动编写大量提示词
- 缺少专业领域的标准化技能模板
- 不同岗位的工作难以标准化和自动化
- 无法快速为Claude添加专业能力

# 主要功能
- 适用于Claude Code和Claude AI的专业技能集合
- 包含多类岗位的Agent技能
- HR板块包含18个技能，支持KPI设计、绩效评估、晋升建议
- 可直接在Claude Code中安装使用
- 提供标准化的专业工作流程

# 项目亮点

- HR功能特别强大，包含18个技能

# 安装或部署教程

## 教程适用对象与环境范围
本教程适用于想要使用claude-skills提升Claude Code或Claude AI能力的用户。

## 前置条件检查

### 1. Claude Code检查
- 步骤目标：确认已安装Claude Code
- 执行位置：本机
- 检查方法：确认Claude Code已正确安装并可正常使用
- 成功标志：Claude Code可正常启动和使用
- 失败处理：请先安装Claude Code

### 2. 网络访问检查
- 步骤目标：确认可以访问GitHub
- 执行位置：本机浏览器
- 检查方法：访问https://github.com/alirezarezvani/claude-skills
- 成功标志：可以正常访问项目页面
- 失败处理：请配置海外网络环境 [海外网络环境配置 →](/网络环境)

## 主路径：安装与使用

### 步骤1：安装Claude Code插件
- 步骤目标：在Claude Code中安装插件
- 执行位置：Claude Code终端
- 完整命令：`/plugin marketplace add alirezarezvani/claude-skills`
- 成功标志：插件安装成功
- 失败处理：检查Claude Code版本和网络连接

### 步骤2：安装技能包
- 步骤目标：安装需要的技能包
- 执行位置：Claude Code终端
- 完整命令：`/plugin install marketing-`（根据需要选择技能）
- 成功标志：技能包安装成功
- 失败处理：查看可用的技能包列表

### 步骤3：使用技能
- 步骤目标：开始使用已安装的技能
- 执行位置：Claude Code
- 使用方法：在对话中直接使用相应技能
- 成功标志：Claude能够使用专业技能完成任务
- 失败处理：确认技能已正确安装

## 安装成功验证

### 1. 测试HR技能
- 测试方法：尝试使用HR相关技能，如KPI设计
- 预期返回：Claude能够使用专业技能完成任务

### 2. 查看可用技能
- 测试方法：查看已安装的技能列表
- 预期返回：显示所有已安装的技能

## 常见错误排查

### 现象1：插件安装失败
- 可能原因：网络问题或Claude Code版本不兼容
- 解决步骤：
  1. 检查网络连接
  2. 配置海外网络环境 [海外网络环境配置 →](/网络环境)
  3. 更新Claude Code到最新版本
  4. 重新尝试安装

### 现象2：技能无法使用
- 可能原因：技能未正确安装或Claude Code配置问题
- 解决步骤：
  1. 确认技能已正确安装
  2. 重启Claude Code
  3. 重新安装技能

### 现象3：Claude Code无法访问
- 可能原因：网络问题或权限问题
- 解决步骤：
  1. 检查网络连接
  2. 确认有使用Claude Code的权限
  3. 尝试重新登录Claude Code
