# 项目名称
RuView

# 作者或组织名
ruvnet

# 项目类别
AI工具

# 核心痛点
传统人体姿态识别依赖摄像头，存在隐私问题；需要穿戴设备，使用不便。

# 主要功能
用普通WiFi信号实现实时人体姿态估计，不需要摄像头和穿戴设备。

# 项目亮点
1. 无需摄像头，保护隐私
2. 无需穿戴设备，使用方便
3. 用Rust语言编写，性能强悍
4. 实时人体姿态估计
5. 通过分析WiFi信号变化提取人体运动信息
