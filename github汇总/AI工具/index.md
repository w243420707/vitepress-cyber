# 🤖 AI工具

## 概述

AI 工具是人工智能时代的重要生产力工具。本分类汇集了各类优质的开源 AI 工具，涵盖大语言模型、AI 助手、图像生成、视频生成等，帮助您提升工作效率。

## 工具列表

### Ollama

- **项目名称**: Ollama
- **作者**: Ollama
- **核心功能**: 在本地电脑上轻松运行大语言模型，超简单的命令行工具，一条命令就能下载和运行模型
- **主要亮点**:
  - 超简单的命令行工具，一条命令就能下载和运行模型
  - 支持超多模型（Llama 3、Phi 3、Mistral、Gemma 等）
  - 跨平台支持（macOS、Windows、Linux）
  - 安装超简单，下载安装包或用 Docker
  - 支持自定义模型，可导入 GGUF 模型
  - 提供 REST API，可在应用中集成
  - 支持 Web UI，方便在浏览器中使用
  - 保护隐私，数据不离开本地
  - 最受欢迎的本地 LLM 运行工具之一
- **详细介绍**: [Ollama 完整文档](/github汇总/AI工具/Ollama-AI工具)
- **GitHub 地址**: https://github.com/ollama/ollama
