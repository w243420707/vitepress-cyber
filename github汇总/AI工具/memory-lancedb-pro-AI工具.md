# 项目名称
memory-lancedb-pro

# 作者或组织名
CortexReach

# 项目类别
AI工具

# 核心痛点
- AI Agent记性差，容易遗忘重要信息
- 缺少可靠的长期记忆解决方案
- 记忆检索效果不佳
- 无法智能提取和分类记忆内容

# 主要功能
- 为AI Agent提供真正的长期记忆能力
- 支持向量搜索和BM25全文检索
- 混合融合和跨编码器Rerank
- LLM智能提取6个类别
- 是OpenClaw自研的LanceDB插件

# 项目亮点
- 为Agent提供真正的长期记忆能力
- 结合向量搜索和全文检索，效果更好
- 支持混合融合和跨编码器重排序
- LLM智能提取和分类记忆内容
- OpenClaw自研插件，兼容性好

## 常见错误排查

### 现象1：插件无法安装
- 可能原因：OpenClaw版本不兼容或网络问题
- 解决步骤：
  1. 确认OpenClaw版本支持该插件
  2. 检查网络连接
  3. 查看项目Issue页面是否有类似问题

### 现象2：记忆检索失效
- 可能原因：配置问题或插件故障
- 解决步骤：
  1. 检查插件配置是否正确
  2. 尝试重新安装插件
  3. 查看项目文档和Issue

### 现象3：性能问题
- 可能原因：记忆数据过大或索引问题
- 解决步骤：
  1. 检查记忆数据量
  2. 优化配置参数
  3. 查看项目文档了解性能优化建议
