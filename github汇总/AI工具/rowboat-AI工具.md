# 项目名称
Rowboat

# 作者或组织名
Rowboat Labs

# 项目类别
AI工具

# 核心痛点
工作中需要处理大量的电子邮件和会议笔记，难以保持对重要信息的记忆和理解，需要一个能够帮助整理和利用这些信息的工具。

# 主要功能
- 连接到电子邮件和会议笔记，构建长期知识图谱
- 生成基于知识图谱的演示文稿、会议简报等
- 可视化、编辑和更新知识图谱
- 录制语音备忘录，自动捕获和更新知识图谱中的关键要点
- 支持本地模型和托管模型
- 通过Model Context Protocol (MCP)连接到外部工具和服务

# 项目亮点
- 本地优先设计，所有数据以纯Markdown格式存储在用户机器上
- 没有专有格式或托管锁定，用户可以随时检查、编辑、备份或删除所有内容
- 支持多种模型，包括本地模型和托管模型
- 可以通过MCP连接到外部工具和服务
- 背景代理可以自动执行重复性工作

# 安装或部署教程

## 教程适用对象与环境范围
适用于需要在工作中管理大量信息和提高工作效率的用户，支持Mac、Windows和Linux系统。

## 前置条件检查
1. 系统要求：Mac、Windows或Linux
2. 依赖：无特殊依赖
3. 权限要求：有足够的权限安装软件

## 主路径：安装与启动

### 下载安装
1. **下载最新版本**
   - 执行位置：浏览器
   - 操作：访问 https://github.com/rowboatlabs/rowboat/releases/latest 下载对应系统的安装包
   - 成功标志：下载完成，获得安装包
   - 失败处理：检查网络连接

2. **安装软件**
   - 执行位置：本地
   - 操作：运行下载的安装包，按照提示完成安装
   - 成功标志：安装完成，软件可以启动
   - 失败处理：检查系统权限，确保有足够的权限安装软件

### Google设置
1. **连接Google服务**
   - 执行位置：Rowboat应用
   - 操作：按照Google设置指南连接Gmail、Calendar和Drive
   - 成功标志：Google服务连接成功
   - 失败处理：检查Google账户权限

### 语音笔记设置
1. **添加Deepgram API密钥**
   - 执行位置：本地
   - 操作：在 ~/.rowboat/config/deepgram.json 文件中添加Deepgram API密钥
     ```json
     {
       "apiKey": "<key>"
     }
     ```
   - 成功标志：文件创建成功
   - 失败处理：检查文件路径是否正确

### 网络搜索设置
1. **添加Brave搜索API密钥**
   - 执行位置：本地
   - 操作：在 ~/.rowboat/config/brave-search.json 文件中添加Brave API密钥
   - 成功标志：文件创建成功
   - 失败处理：检查文件路径是否正确

2. **添加Exa研究搜索API密钥**
   - 执行位置：本地
   - 操作：在 ~/.rowboat/config/exa-search.json 文件中添加Exa API密钥
   - 成功标志：文件创建成功
   - 失败处理：检查文件路径是否正确

## 安装成功验证
- 执行操作：启动Rowboat应用
- 预期返回：应用启动成功，显示主界面

## 常见错误排查
1. **Google服务连接失败**
   - 现象：无法连接Gmail、Calendar或Drive
   - 可能原因：Google账户权限不足、网络连接问题
   - 解决步骤：检查Google账户权限，确保网络连接正常

2. **语音笔记功能无法使用**
   - 现象：无法录制语音备忘录
   - 可能原因：Deepgram API密钥未正确配置
   - 解决步骤：检查Deepgram API密钥是否正确配置

3. **网络搜索功能无法使用**
   - 现象：无法使用Brave或Exa搜索
   - 可能原因：API密钥未正确配置
   - 解决步骤：检查API密钥是否正确配置
