# 项目名称
notebooklm-py

# 作者或组织名
teng-lin

# 项目类别
AI工具

# 核心痛点
NotebookLM只能通过网页访问，缺乏命令行接口，难以集成到自动化工作流中。

# 主要功能
NotebookLM的Python命令行工具，逆向工程了NotebookLM未记录的API，提供对平台功能的访问。

# 项目亮点
1. NotebookLM的Python命令行工具
2. 逆向工程了NotebookLM未记录的API
3. 为用户提供对平台功能的访问
4. 支持创建笔记本
5. 支持添加来源（URL、PDF、YouTube等）
6. 可以通过自然语言指令直接调用
