# 项目名称
Moonshine Voice

# 作者或组织名
Moonshine AI

# 项目类别
AI工具

# 核心痛点
实时语音应用需要低延迟响应，而现有解决方案如Whisper存在固定输入窗口、不支持缓存、多语言支持差等问题。

# 主要功能
开源AI工具包，用于构建实时语音应用，支持多平台，提供低延迟响应，内置语音识别、命令识别等功能。

# 项目亮点
- 设备本地运行：速度快、隐私性好，无需API密钥
- 低延迟响应：在用户说话时完成大量工作
- 多模型支持：从26MB小模型到高精度模型
- 跨平台集成：支持多种操作系统和设备
- 内置高级API：提供完整解决方案
- 多语言支持：英语、西班牙语、普通话等多种语言

# 安装或部署教程

## 教程适用对象与环境范围
适用于需要构建实时语音应用的开发者，支持多种平台。

## 前置条件检查
1. Python 3.7+（Python用户）
2. 支持的平台：Linux、macOS、Windows、iOS、Android、树莓派等
3. 麦克风（用于实时语音输入）

## 主路径：安装与使用
1. **Python安装**
   - 目标：安装Moonshine Voice Python包
   - 位置：命令行
   - 命令：`pip install moonshine-voice`
   - 成功标志：安装完成，无错误信息
   - 失败处理：检查Python版本是否符合要求

2. **下载模型**
   - 目标：下载语音模型
   - 位置：命令行
   - 命令：`python -m moonshine_voice.download --language en`
   - 成功标志：模型下载完成，显示模型路径和架构
   - 失败处理：检查网络连接，确保有足够的磁盘空间

3. **测试麦克风转录**
   - 目标：测试实时语音转录功能
   - 位置：命令行
   - 命令：`python -m moonshine_voice.mic_transcriber --language en`
   - 成功标志：开始监听麦克风，实时显示转录结果
   - 失败处理：检查麦克风权限，确保麦克风正常工作

4. **测试命令识别**
   - 目标：测试语音命令识别功能
   - 位置：命令行
   - 命令：`python -m moonshine_voice.intent_recognizer`
   - 成功标志：开始监听，识别预设命令
   - 失败处理：检查麦克风权限，确保模型正确加载

## 安装成功验证
- 访问地址：N/A（本地工具）
- 检查命令：运行测试命令
- 预期返回：实时显示语音转录结果或命令识别结果

## 常见错误排查
1. **模型下载失败**
   - 现象：下载模型时出错
   - 可能原因：网络连接问题，或磁盘空间不足
   - 解决步骤：检查网络连接，确保有足够的磁盘空间

2. **麦克风权限问题**
   - 现象：无法访问麦克风
   - 可能原因：未授予麦克风权限
   - 解决步骤：在系统设置中授予麦克风权限

3. **模型加载失败**
   - 现象：运行时提示模型加载失败
   - 可能原因：模型路径错误，或模型文件损坏
   - 解决步骤：检查模型路径是否正确，重新下载模型

4. **平台兼容性问题**
   - 现象：在特定平台上运行失败
   - 可能原因：平台不支持，或依赖项缺失
   - 解决步骤：检查平台兼容性，安装必要的依赖项
