# 项目类别
AI工具

# 核心痛点
- 多平台内容发布需要手动操作，效率低
- 视频扒字幕需要多个工具
- AI润色和多平台分发需要分开处理
- 缺少一站式多平台集成工具

# 主要功能
- 一条命令给AI员工打通7大平台
- 视频一键扒字幕（YouTube、TikTok、B站）
- 一键AI润色
- 多平台一键发布
- 集成7大常用社交平台（YouTube、TikTok、Instagram、Twitter/X、小红书、B站）
- 让内容快速触达更多用户

## 常见错误排查

### 现象1：工具无法安装
- 可能原因：OpenClaw版本不兼容或网络问题
- 解决步骤：
  1. 确认OpenClaw版本支持该工具
  2. 检查网络连接
  3. 查看项目Issue页面是否有类似问题

### 现象2：平台接入失败
- 可能原因：平台配置问题或权限问题
- 解决步骤：
  1. 检查平台配置是否正确
  2. 确认有足够的权限
  3. 查看项目文档了解平台接入要求
