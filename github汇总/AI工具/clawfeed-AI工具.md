# 项目名称
clawfeed

# 作者或组织名
OpenClawFeed

# 项目类别
AI工具

# 核心痛点
- 需要手动查找内容，效率低
- 缺少AI主动订阅和推送功能
- 内容获取不够智能和个性化
- OpenClaw生态缺少内容订阅功能

# 主要功能
- OpenClaw的内容订阅和推送功能
- 让AI员工主动获取和推送内容
- 根据订阅偏好主动获取感兴趣的内容
- OpenClaw生态的重要组成部分
- 让AI员工更加智能和主动

# 项目亮点

- 个性化订阅偏好

## 常见错误排查

### 现象1：工具无法安装
- 可能原因：OpenClaw版本不兼容或网络问题
- 解决步骤：
  1. 确认OpenClaw版本支持该工具
  2. 检查网络连接
  3. 查看项目Issue页面是否有类似问题

### 现象2：内容推送不工作
- 可能原因：配置问题或网络问题
- 解决步骤：
  1. 检查订阅配置是否正确
  2. 检查网络连接
  3. 查看项目文档和Issue
