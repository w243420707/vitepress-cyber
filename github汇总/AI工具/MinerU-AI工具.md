# 项目名称
MinerU

# 作者或组织名
OpenDataLab

# 项目类别
AI工具

# 核心痛点
文档提取困难，PDF、网页、多格式电子书等文档格式转换繁琐，缺乏一站式开源解决方案。

# 主要功能
一站式、开源、高质量的数据提取工具，支持PDF、网页、多格式电子书提取并转换为Markdown格式，包含Magic-PDF和Magic-Doc两个核心功能模块。

# 项目亮点
- 支持多种文档格式（PDF、网页、电子书）
- 一站式开源解决方案
- 高质量数据提取
- 转换为清晰的Markdown格式
- 包含Magic-PDF和Magic-Doc两个核心模块
