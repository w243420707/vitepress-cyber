# 项目名称
TradingAgents-CN

# 作者或组织名
hsliuping

# 项目类别
AI工具

# 核心痛点
个人投资者缺乏专业的交易分析能力，难以获取全面的市场信息，风险评估能力有限，缺乏系统的投资策略框架。

# 主要功能
- 专业级交易分析
- 实时行情监控
- 多维度风险评估
- 多样化投资策略
- A股、港股、美股多市场覆盖
- 股票分析、期货研究、宏观分析、实时资讯、深度研报解读
- 三个版本：基础简易版、进阶专业版、生产企业版
- Docker支持

# 项目亮点
- 基于LangGraph的智能体架构
- 中文本地化支持
- 小白友好，支持Docker快速部署
- 结构化的分析框架
- 多市场、多维度分析能力

# 安装或部署教程

## 教程适用对象与环境范围
适用于需要智能交易分析的交易者和投资者，支持Docker部署。

## 前置条件检查
1. Docker环境：已安装Docker和Docker Compose
2. API密钥：需要配置相关交易数据API密钥

## 主路径：Docker部署（推荐）

### 步骤1：克隆项目
步骤目标：获取项目代码
执行位置：终端
完整命令或操作：
```bash
git clone https://github.com/hsliuping/TradingAgents-CN.git
cd TradingAgents-CN
```
成功标志：克隆成功，进入项目目录
失败处理：检查网络连接，确认Git安装

### 步骤2：配置环境
步骤目标：配置API密钥和环境变量
执行位置：项目目录
完整命令或操作：复制配置模板，编辑配置文件
成功标志：配置文件编辑完成
失败处理：查看项目文档确认配置项

### 步骤3：启动服务
步骤目标：使用Docker启动服务
执行位置：项目目录
完整命令或操作：
```bash
docker-compose up -d
```
成功标志：Docker容器成功启动
失败处理：查看Docker日志，检查端口占用

## 安装成功验证
1. Docker容器正常运行
2. 可以访问Web界面（如有）
3. 能成功获取市场数据

## 常见错误排查

1. **现象 -> Docker启动失败**
   可能原因 -> Docker环境问题或配置错误
   解决步骤 -> 检查Docker服务状态，查看容器日志

2. **现象 -> API调用失败**
   可能原因 -> API密钥配置错误或权限问题
   解决步骤 -> 检查API密钥配置，确认API服务可用性

3. **现象 -> 数据获取异常**
   可能原因 -> 网络问题或数据源限制
   解决步骤 -> 检查网络连接，确认数据源配额
