# LangGPT

## 项目名称
LangGPT

## 作者或组织名
langgptai

## 项目类别
AI工具

## 核心痛点
传统提示词工程依赖零散技巧和反复试错，缺乏系统化的方法论。

## 主要功能
结构化可复用的提示词设计框架，让每个人都能创建高质量的AI提示词。采用类似编程语言的设计思路，提供结构化模板、可复用性、模块化和高效率四大核心特性。通过角色定义（Role）、档案配置（Profile）、技能设定（Skills）、规则约束（Rules）和工作流程（Workflow）五个模块构建完整提示词。支持变量引用、命令定义和条件逻辑等高级功能。提供丰富的示例库和多种AI模型的专门提示词集合。

## 安装或部署教程

### 教程适用对象与环境范围
本教程适用于想要使用LangGPT框架编写提示词的用户。

### 主路径：使用在线工具（推荐）

**步骤1：访问LangGPT GPTs**
- 目标：使用AI自动生成提示词
- 位置：浏览器
- 操作：访问 LangGPT GPTs（需要GPT-4访问权限）
- 成功标志：可以与GPTs对话生成提示词
- 失败处理：确认有GPT-4访问权限

**步骤2：描述需求**
- 目标：让AI生成结构化提示词
- 位置：GPTs对话界面
- 操作：描述你想要的AI角色和功能
- 成功标志：AI生成符合LangGPT格式的提示词
- 失败处理：提供更详细的需求描述

### 备选路径：手动学习模板

**步骤1：访问项目文档**
- 目标：学习LangGPT模板结构
- 位置：浏览器
- 操作：访问 https://github.com/langgptai/LangGPT 查看文档
- 成功标志：理解模板结构
- 失败处理：查看示例库

**步骤2：查看示例**
- 目标：学习优秀提示词案例
- 位置：项目示例库
- 操作：浏览Examples目录下的提示词示例
- 成功标志：理解如何编写结构化提示词
- 失败处理：参考视频教程

### Claude Code集成

**步骤1：下载技能文件**
- 目标：获取LangGPT技能文件
- 位置：项目仓库
- 操作：下载 langgpt.skill 文件
- 成功标志：文件下载完成
- 失败处理：检查网络连接

**步骤2：安装技能**
- 目标：将技能安装到Claude Code
- 位置：本地
- 操作：将文件解压到 ~/.claude/skills/ 目录
- 成功标志：技能安装成功
- 失败处理：确认目录路径正确

**步骤3：使用技能**
- 目标：在Claude Code中使用LangGPT
- 位置：Claude Code
- 操作：输入 /langgpt 命令
- 成功标志：技能激活，可以生成提示词
- 失败处理：确认技能正确安装

### 常见错误排查

1. **GPTs无法访问**
   - 可能原因：没有GPT-4订阅
   - 解决步骤：升级ChatGPT订阅或使用手动模板

2. **Claude Code技能不生效**
   - 可能原因：技能文件路径错误
   - 解决步骤：确认文件放在正确的 ~/.claude/skills/ 目录

3. **生成的提示词效果不好**
   - 可能原因：需求描述不够清晰
   - 解决步骤：提供更详细的角色定义和功能需求
