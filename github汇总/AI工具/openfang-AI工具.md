# 项目名称
OpenFang

# 作者或组织名
RightNow AI

# 项目类别
AI工具

# 核心痛点
AI代理只能被动聊天，无法长期自主运行，碎片化能力难以整合。

# 主要功能
RightNow AI开源的生产级AgentOS，基于Rust开发，将AI代理视为一等公民的进程，把碎片化能力串成可长期运行的自动化流程。

# 项目亮点
1. 生产级AgentOS
2. 基于Rust开发
3. 将AI代理视为一等公民的进程
4. 把碎片化能力串成可长期运行的自动化流程
5. 从被动聊天转向后台自主运行
6. 上线仅4天就获得大量星标
