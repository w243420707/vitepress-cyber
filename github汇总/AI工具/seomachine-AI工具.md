# 项目名称
SEO Machine

# 作者或组织名
jiyangliu

# 项目类别
AI工具

# 核心痛点
独立站和电商商家需要SEO写作工具，但OpenAI API Key获取困难。

# 主要功能
专门为独立站和电商商家设计的AI SEO写作工具，核心优势是无需OpenAI API Key即可使用Gemini API。

# 项目亮点
1. 无需OpenAI API Key即可使用Gemini API
2. 专门为独立站和电商商家设计
3. 集成了SEO写作、HTML嵌入等功能
4. 本地HTML工具，开箱即用
5. 零代码基础也能快速上手
6. 有实时预览、一键保存等辅助功能

# 安装或部署教程

## 1. 教程适用对象与环境范围
适用于独立站和电商商家，支持Windows、macOS、Linux系统。

## 2. 前置条件检查
- 系统版本：Windows、macOS或Linux
- 浏览器：现代浏览器（Chrome、Firefox、Edge等）
- 网络要求：能够访问GitHub和Gemini API
- 检查命令：无特殊要求

## 3. 主路径：安装与使用
### 步骤1：克隆仓库
- 步骤目标：克隆项目仓库到本地
- 执行位置：本机（终端/命令提示符）
- 完整命令或操作：
```bash
git clone https://github.com/jiyangliu/seomachine.git
cd seomachine
```
- 成功标志：仓库克隆成功，进入项目目录
- 失败处理：检查网络连接，或直接从GitHub下载ZIP压缩包

### 步骤2：打开工具
- 步骤目标：打开本地HTML工具
- 执行位置：项目目录
- 完整命令或操作：直接用浏览器打开HTML文件
- 成功标志：工具在浏览器中正常打开
- 失败处理：检查HTML文件完整性

## 4. 安装成功验证
- 访问地址：本地HTML文件路径
- 检查命令：无
- 预期返回：工具界面正常显示

## 5. 常见错误排查
1. 现象：git clone 失败
   - 可能原因：网络问题
   - 解决步骤：检查网络连接，或直接下载ZIP文件
2. 现象：工具无法打开
   - 可能原因：浏览器不兼容
   - 解决步骤：使用现代浏览器（Chrome、Firefox、Edge）
3. 现象：Gemini API调用失败
   - 可能原因：网络问题
   - 解决步骤：检查网络连接
