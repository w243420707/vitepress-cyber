# 项目名称
xiaohongshu-mcp

# 作者或组织名
xpzouying

# 项目类别
AI工具

# 核心痛点
小红书运营需要大量时间和精力，内容生产效率低，缺乏自动化工具。

# 主要功能
基于Model Context Protocol (MCP)的小红书AI自动化运营工具，支持登录状态保持、图文发布、内容搜索等功能。

# 项目亮点
1. 基于Model Context Protocol (MCP)
2. 通过浏览器自动化模拟人工操作
3. 支持登录状态保持、图文发布、内容搜索、写评论
4. 号称可以日更50篇不封号
5. 实现AI助手与小红书平台的深度集成
