# 项目名称
chatgpt-on-wechat

# 作者或组织名
zhayujie

# 项目类别
AI工具

# 核心痛点
想在微信上使用ChatGPT，但缺乏便捷的集成方式

# 主要功能
- 基于ChatGPT的微信机器人
- 支持多种大语言模型（GPT-4、Claude、Gemini、通义千问等）
- 支持多种消息类型（文字、图片、语音、文件等）
- 支持图片生成、语音回复等功能
- 可通过插件系统扩展功能

# 项目亮点
- 完全免费，代码开源
- 支持多种对接方式，避免封号风险
- 支持多种主流大模型
- 功能丰富，扩展性强
