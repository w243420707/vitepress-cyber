# 项目名称
BettaFish

# 作者或组织名
666ghj

# 项目类别
AI工具

# 核心痛点
信息茧房问题严重，用户难以获取全面客观的舆情信息；舆情分析成本高，需要大量人工处理。

# 主要功能
多智能体舆情分析系统，帮助破除信息茧房，还原舆情原貌，预测未来走向。

# 项目亮点
1. 采用多智能体架构
2. 宣称可处理约210万有效文本、约3.8亿次阅读量、约420万次互动
3. 支持Windows、Linux、MacOS多操作系统
4. 采用GPL-2.0许可证
5. 适用于品牌声量分析、竞品对比等场景
