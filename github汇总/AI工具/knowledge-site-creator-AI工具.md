# 项目名称
knowledge-site-creator

# 作者或组织名
joeseesun

# 项目类别
AI工具

# 核心痛点
- 创建学习网站需要大量代码和设计工作
- 制作Skill门槛高，难以分享
- 缺少自动化的学习内容生成工具
- OpenClaw生态需要更多优质Skill

# 主要功能
- 告诉AI你想学什么，自动生成完整学习网站
- 自动部署学习网站上线
- 快速创建Skill并部署上线
- Skill可以分享给其他人使用
- 完美融入OpenClaw生态系统

# 项目亮点
- 自动化程度高，只需告诉AI想学什么
- 自动生成完整学习网站内容和结构
- 可以创建和分享Skill
- 完美融入OpenClaw生态
- 使用简单，无需编写代码

- 失败处理：请先安装OpenClaw

## 常见错误排查

### 现象1：Skill无法安装
- 可能原因：OpenClaw版本不兼容或网络问题
- 解决步骤：
  1. 确认OpenClaw版本支持该Skill
  2. 检查网络连接
  3. 查看项目Issue页面是否有类似问题

### 现象2：网站生成失败
- 可能原因：AI理解问题或配置问题
- 解决步骤：
  1. 更清晰地描述你想学习的主题
  2. 检查Skill配置
  3. 查看项目文档和Issue

### 现象3：部署失败
- 可能原因：部署配置问题或权限问题
- 解决步骤：
  1. 检查部署配置
  2. 确认有足够的权限
  3. 查看项目文档了解部署要求
