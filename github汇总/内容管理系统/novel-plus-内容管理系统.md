# 项目类别
内容管理系统

# 核心痛点
需要一个功能完善的原创文学CMS系统，支持多端阅读、AI写作等功能。

# 主要功能
多端（PC、WAP）阅读，功能完善的原创文学CMS系统，包括前台门户、作家后台、平台后台和爬虫管理系统。

# 项目亮点
- 多端支持：PC、WAP端阅读
- 完整功能：小说推荐、作品检索、小说排行、小说阅读、小说评论、会员中心、作家专区
- 自定义模板：支持自定义多模版
- 多种存储方式：内置数据库分表存储和TXT文本存储
- 阅读体验：支持阅读主题切换
- 数据采集：多爬虫源自动采集和更新数据
- AI功能：AI扩写、缩写、续写、文本润色、AI生成封面图
- 商业模式：会员充值、订阅模式
- 管理功能：新闻发布和实时统计报表

# 安装或部署教程

## 教程适用对象与环境范围
适用于需要搭建文学网站的开发者和运营者。

## 前置条件检查
1. JDK 8+
2. MySQL 5.7+
3. Redis
4. Maven 3.6+
5. Docker（可选）

## 主路径：安装与使用
1. **克隆代码**
   - 目标：克隆novel-plus项目代码
   - 位置：命令行
   - 命令：
     ```bash
     git clone https://github.com/201206030/novel-plus.git
     cd novel-plus
     ```
   - 成功标志：代码克隆完成
   - 失败处理：检查网络连接，确保GitHub可以访问

2. **配置数据库**
   - 目标：创建数据库并导入初始数据
   - 位置：MySQL命令行或工具
   - 命令：
     ```sql
     CREATE DATABASE novel_plus CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
     ```
   - 成功标志：数据库创建成功
   - 失败处理：检查MySQL服务是否运行，权限是否正确

3. **配置项目**
   - 目标：配置项目的数据库连接和其他参数
   - 位置：项目配置文件
   - 命令：编辑 `novel-front/src/main/resources/application.yml` 文件，配置数据库连接信息
   - 成功标志：配置文件修改完成
   - 失败处理：检查配置文件路径是否正确

4. **构建项目**
   - 目标：构建项目
   - 位置：命令行
   - 命令：
     ```bash
     mvn clean install
     ```
   - 成功标志：构建完成，无错误
   - 失败处理：检查依赖是否正确，网络连接是否正常

5. **启动项目**
   - 目标：启动项目
   - 位置：命令行
   - 命令：
     ```bash
     cd novel-front/target
     java -jar novel-front-*.jar
     ```
   - 成功标志：项目启动成功，可访问 http://localhost:8080
   - 失败处理：检查端口是否被占用，数据库连接是否正确

## 安装成功验证
- 访问地址：http://localhost:8080
- 检查命令：打开浏览器访问上述地址
- 预期返回：novel-plus前台页面

## 常见错误排查
1. **数据库连接失败**
   - 现象：项目启动时无法连接数据库
   - 可能原因：数据库配置错误，或MySQL服务未运行
   - 解决步骤：检查数据库配置，确保MySQL服务运行

2. **端口被占用**
   - 现象：项目启动时端口被占用
   - 可能原因：8080端口已被其他应用占用
   - 解决步骤：修改项目配置，使用其他端口

3. **依赖下载失败**
   - 现象：Maven构建时依赖下载失败
   - 可能原因：网络连接问题，或Maven配置错误
   - 解决步骤：检查网络连接，配置Maven镜像

4. **AI功能无法使用**
   - 现象：AI功能无法正常使用
   - 可能原因：未配置AI API密钥
   - 解决步骤：注册硅基流动账号，获取API密钥并配置到项目中
