# CogVideo

## 作者或组织名
清华大学 THUDM

## 项目类别
AI视频生成

## 核心痛点
视频生成需要付费服务或复杂的模型部署，门槛较高

## 主要功能
- 开源视频生成模型系列
- 支持文生视频、图生视频、视频续写等多种任务
- 多个版本：CogVideoX-2B、CogVideoX-5B、CogVideoX1.5-5B等
- 优化出色，可在普通显卡上运行
- 提供完整的推理和微调代码
- 提供Colab笔记本可免费体验

## 项目亮点
- 清华大学开发，中文社区支持好
- 多个版本选择，从2B到5B参数量
- 支持文生视频、图生视频、视频续写等多种任务
- 最新版本支持10秒高分辨率视频生成
- 优化出色，CogVideoX-2B可在GTX 1080TI运行，CogVideoX-5B可在RTX 3060运行
- 提供完整的推理和微调代码
- 提供Colab笔记本可免费体验
- 活跃的社区和丰富的生态
