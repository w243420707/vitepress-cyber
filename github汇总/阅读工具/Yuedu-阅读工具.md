# 项目名称
Yuedu

# 作者或组织名
XIU2

# 项目类别
阅读工具

# 核心痛点
「阅读」APP用户需要优质的书源来获取小说内容。

# 主要功能
为「阅读」APP提供自用书源分享，包含网上搜集和自己编写的小说书源规则。

# 项目亮点
- 书源丰富：包含网上搜集/他人推荐和自己编写的书源规则
- 多种导入方式：支持二维码导入、网络导入和本地导入
- 使用建议：提供了书源使用的最佳实践
- 持续更新：定期更新书源，确保可用性

# 安装或部署教程

## 教程适用对象与环境范围
适用于「阅读」APP的用户，需要添加优质书源。

## 前置条件检查
1. 安装「阅读」APP
2. 网络连接正常

## 主路径：导入书源
1. **二维码导入（推荐）**
   - 目标：通过二维码导入书源
   - 位置：「阅读」APP内
   - 命令：
     1. 打开「阅读」APP
     2. 点击右下角的「我的」
     3. 点击「书源管理」
     4. 点击右上角的「┇」
     5. 点击「二维码导入」
     6. 扫描项目提供的二维码
   - 成功标志：书源导入成功，可在书源列表中看到
   - 失败处理：检查网络连接，确保二维码清晰

2. **网络导入（推荐）**
   - 目标：通过网络地址导入书源
   - 位置：「阅读」APP内
   - 命令：
     1. 打开「阅读」APP
     2. 点击右下角的「我的」
     3. 点击「书源管理」
     4. 点击右上角的「┇」
     5. 点击「网络导入」
     6. 输入书源地址（任选一个）：
        - https://bitbucket.org/xiu2/yuedu/raw/master/shuyuan
        - https://jsd.onmicrosoft.cn/gh/XIU2/Yuedu/shuyuan
        - https://cdn.jsdmirror.com/gh/XIU2/Yuedu/shuyuan
        - https://wget.la/https://raw.githubusercontent.com/XIU2/Yuedu/master/shuyuan
        - https://ghfast.top/https://raw.githubusercontent.com/XIU2/Yuedu/master/shuyuan
        - https://gh.catmak.name/https://raw.githubusercontent.com/XIU2/Yuedu/master/shuyuan
        - https://cdn.gh-proxy.org/https://raw.githubusercontent.com/XIU2/Yuedu/master/shuyuan
     7. 点击「确定」
   - 成功标志：书源导入成功，可在书源列表中看到
   - 失败处理：检查网络连接，确保书源地址可用

3. **本地导入**
   - 目标：通过本地文件导入书源
   - 位置：「阅读」APP内
   - 命令：
     1. 访问书源分享地址 https://yuedu.xiu2.xyz
     2. 点击「下载文件」下载 shuyuan.json 文件
     3. 打开「阅读」APP
     4. 点击右下角的「我的」
     5. 点击「书源管理」
     6. 点击右上角的「┇」
     7. 点击「本地导入」
     8. 选择下载的 shuyuan.json 文件
   - 成功标志：书源导入成功，可在书源列表中看到
   - 失败处理：检查文件路径是否正确，确保文件未损坏

## 安装成功验证
- 访问地址：「阅读」APP → 书源管理
- 检查命令：查看书源列表是否包含导入的书源
- 预期返回：书源列表中显示导入的书源

## 常见错误排查
1. **书源导入失败**
   - 现象：书源导入后无法使用
   - 可能原因：网络连接问题，或书源地址失效
   - 解决步骤：检查网络连接，尝试使用其他书源地址

2. **书源失效**
   - 现象：书源无法获取小说内容
   - 可能原因：小说网站结构变化，或书源规则过时
   - 解决步骤：更新书源，或尝试其他书源

3. **访问限制**
   - 现象：书源访问失败，提示验证码或无法连接
   - 可能原因：触发了小说网站的访问速率限制
   - 解决步骤：在「阅读」APP的其他设置中将预下载和更新搜索线程数调低（建议1~5）

4. **听书功能问题**
   - 现象：无法使用听书功能
   - 可能原因：手机自带TTS不支持中文
   - 解决步骤：安装额外的TTS引擎，如TTS Server、MultiTTS、微软TTS等

## 备选部署路径
- **源仓库**：去源仓库或Yiove综合书源库获取更多书源
- **小说下载器**：使用开源的小说下载器下载小说
