# 项目名称
sioyek

# 作者或组织名
ahrm

# 项目类别
阅读工具

# 核心痛点
学术PDF阅读体验不好，缺乏专为科研人员设计的高效PDF阅读器。

# 主要功能
专为科研人员设计的开源PDF查看器，专注于提升技术文献阅读效率，支持键盘快捷键和多种导航功能，GitHub Star数突破17000。

# 项目亮点
- 专为科研人员设计的开源PDF查看器
- 专注于提升技术文献阅读效率
- 支持键盘快捷键
- 支持多种导航功能
- GitHub Star数突破17000
- 支持搜索、跳转、书签等功能
