# 项目名称
阅读3.0 (legado)

# 作者或组织名
gedoor

# 项目类别
阅读工具

# 核心痛点
用户需要一个功能丰富、无广告、支持自定义书源的阅读应用。

# 主要功能
开源阅读应用，支持自定义书源、本地阅读、高度自定义阅读界面等功能。

# 项目亮点
- 自定义书源：支持自己设置规则，抓取网页数据
- 书架模式：列表和网格书架自由切换
- 书源搜索：支持搜索及发现功能
- 内容订阅：可以订阅想看的任何内容
- 替换净化：去除广告替换内容
- 本地阅读：支持TXT、EPUB格式
- 高度自定义：字体、颜色、背景、行距等
- 多种翻页模式：覆盖、仿真、滑动、滚动等
- 开源无广告：持续优化，无广告

# 安装或部署教程

## 教程适用对象与环境范围
适用于喜欢阅读的用户，需要一个功能丰富、无广告的阅读应用。

## 前置条件检查
1. Android设备
2. 允许安装来自未知来源的应用

## 主路径：安装与使用
1. **下载安装包**
   - 目标：下载阅读3.0的APK安装包
   - 位置：GitHub Releases页面
   - 命令：访问https://github.com/gedoor/legado/releases下载最新版本的APK
   - 成功标志：下载完成
   - 失败处理：检查网络连接，确保GitHub可以访问

2. **安装应用**
   - 目标：安装阅读3.0应用
   - 位置：Android设备
   - 命令：打开下载的APK文件，按照提示安装
   - 成功标志：应用安装完成，图标出现在桌面
   - 失败处理：检查设备是否允许安装来自未知来源的应用

3. **添加书源**
   - 目标：添加自定义书源
   - 位置：应用内
   - 命令：打开应用 → 书源管理 → 右上角添加 → 输入书源规则
   - 成功标志：书源添加成功，可在书源列表中看到
   - 失败处理：检查书源规则格式是否正确

4. **导入书籍**
   - 目标：导入书籍到书架
   - 位置：应用内
   - 命令：
     - 在线书籍：书源搜索 → 选择书籍 → 加入书架
     - 本地书籍：本地书籍 → 扫描 → 选择书籍 → 加入书架
   - 成功标志：书籍出现在书架中
   - 失败处理：检查书源是否正常，本地书籍路径是否正确

5. **自定义阅读设置**
   - 目标：自定义阅读界面
   - 位置：应用内
   - 命令：打开书籍 → 点击屏幕中央 → 右下角设置 → 调整字体、颜色、背景等
   - 成功标志：阅读界面按照设置显示
   - 失败处理：检查设置是否保存成功

## 安装成功验证
- 访问地址：应用图标
- 检查命令：打开应用，添加书源，导入书籍
- 预期返回：应用正常运行，书籍可正常阅读

## 常见错误排查
1. **书源添加失败**
   - 现象：书源添加后无法使用
   - 可能原因：书源规则格式错误，或网站结构变化
   - 解决步骤：检查书源规则，更新规则以适应网站变化

2. **本地书籍扫描失败**
   - 现象：本地书籍无法扫描到
   - 可能原因：文件路径权限问题，或文件格式不支持
   - 解决步骤：检查存储权限，确保文件格式为TXT或EPUB

3. **阅读界面设置不生效**
   - 现象：调整阅读设置后界面无变化
   - 可能原因：设置未保存，或应用缓存问题
   - 解决步骤：重新调整设置并保存，或清除应用缓存

4. **应用崩溃**
   - 现象：应用启动或使用过程中崩溃
   - 可能原因：设备系统版本不兼容，或应用版本问题
   - 解决步骤：更新应用到最新版本，或检查设备系统版本
