# 项目名称
martin

# 作者或组织名
maplibre

# 项目类别
地图工具

# 核心痛点
地图服务性能不够快，缺乏高性能的开源地图瓦片服务器。

# 主要功能
由MapLibre团队用Rust语言打造的超快速轻量级地理空间地图瓦片服务器，支持从PostgreSQL数据库即时生成矢量瓦片，还支持PMTiles和MBTiles文件服务。

# 项目亮点
- 由MapLibre团队用Rust语言打造
- 超快速轻量级地理空间地图瓦片服务器
- 支持从PostgreSQL数据库即时生成矢量瓦片
- 支持PMTiles文件服务
- 支持MBTiles文件服务
- 官方宣传其性能快到疯狂
- 是当前开源GIS领域的高性能选择
