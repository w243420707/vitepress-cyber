# 项目名称
TikTokDownloader

# 作者或组织名
JoeanAmier

# 项目类别
视频工具

# 核心痛点
需要批量采集和下载抖音和TikTok内容，但缺乏便捷高效的工具

# 主要功能
- 支持下载抖音和TikTok的视频、图集、实况、动图等
- 批量下载账号发布、喜欢、收藏的作品
- 获取直播拉流地址，下载直播视频
- 采集评论数据、账号详细数据、搜索结果、热榜数据等
- 支持多种数据格式保存，多线程下载，断点续传

# 项目亮点
- 完全开源，开箱即用
- 功能全面，涵盖数据采集和下载
- 支持批量操作，效率高
- 提供Web API调用，易于集成
