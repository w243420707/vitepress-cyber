# 项目名称
GitNexus

# 作者或组织名
abhigyanpatwari

# 项目类别
开发工具

# 核心痛点
AI编码助手缺乏代码库的全局视野，容易改一处崩全局；传统代码分析需要服务器，部署复杂。

# 主要功能
把任意代码仓库变成一个可查询的知识图谱，让AI编码助手拥有代码库的全局视野。

# 项目亮点
1. 零服务器架构，完全在浏览器端运行
2. 把代码仓库变成交互式知识图谱
3. 让AI编码助手拥有代码库全局视野
4. 主打深层上下文的基建工具
5. 在线体验，无需安装
