# 安装或部署教程

## 1. 教程适用对象与环境范围
适用于各类开发者，支持Windows、macOS、Linux系统。

## 2. 前置条件检查
- 系统版本：Windows、macOS或Linux
- Node.js版本：Node.js 16或更高版本
- 依赖版本：npm包管理器
- 网络要求：能够访问GitHub和AI服务
- 检查命令：
  - 检查Node.js版本：node --version
  - 检查npm版本：npm --version

## 3. 主路径：安装与启动
### 步骤1：安装codebuff
- 步骤目标：安装codebuff工具
- 执行位置：本机（终端/命令提示符）
- 完整命令或操作：
```bash
npm install -g codebuff
```
- 成功标志：命令执行完成，无错误提示
- 失败处理：如果提示权限不足，尝试使用管理员权限运行

### 步骤2：启动codebuff
- 步骤目标：启动codebuff
- 执行位置：本机（终端/命令提示符）
- 完整命令或操作：
```bash
codebuff
```
- 成功标志：进入codebuff交互界面
- 失败处理：检查Node.js环境和PATH配置

## 4. 安装成功验证
- 访问地址：无（这是一个终端工具，没有Web界面）
- 检查命令：运行 codebuff 命令
- 预期返回：进入codebuff交互界面

## 5. 常见错误排查
1. 现象：npm install -g codebuff 失败
   - 可能原因：网络问题或权限问题
   - 解决步骤：检查网络连接，或使用国内npm镜像源
2. 现象：codebuff 命令未找到
   - 可能原因：PATH配置问题
   - 解决步骤：将npm全局安装目录添加到PATH中
3. 现象：AI功能无法使用
   - 可能原因：AI服务配置问题
   - 解决步骤：检查AI服务API密钥配置
