# 项目名称
JeecgBoot

# 作者或组织名
jeecgboot

# 项目类别
开发工具

# 核心痛点
企业级应用开发中重复工作多，开发效率低，需要兼顾快速搭建和深度定制的需求

# 主要功能
- 低代码和零代码双模式，支持一句话生成完整系统
- 代码生成模式自动输出前后端代码、建表SQL与菜单权限
- 内置AI聊天助手、知识库、流程编排、MCP与插件体系
- 支持一句话生成流程图、一句话设计表单、聊天式业务操作
- 采用前后端分离技术栈，包括Ant Design、Vue3、SpringBoot3等

# 项目亮点
- 零代码模式一句话搭建完整业务系统
- 支持多种主流大模型，智能功能丰富
- 解决Java项目中80%的重复工作
- 支持信创国产化，适用多种企业场景
