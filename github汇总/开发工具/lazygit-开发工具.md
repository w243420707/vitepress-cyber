# 项目名称
lazygit

# 作者或组织名
jesseduffield

# 项目类别
开发工具

# 核心痛点
Git命令记不住操作繁琐，缺乏可视化的终端Git操作工具。

# 主要功能
用Go语言开发的简单而强大的终端UI Git工具，提供可视化交互界面，通过快捷键完成Git操作，完全键盘驱动，是开源世界中最受欢迎的终端Git工具之一，总Star超过63k。

# 项目亮点
- 用Go语言开发
- 简单而强大的终端UI Git工具
- 提供可视化交互界面
- 通过快捷键完成Git操作
- 完全键盘驱动
- 支持提交操作
- 支持推送操作
- 支持分支切换
- 支持rebase操作
- 支持stash操作
- 无需鼠标点击
- 是开源世界中最受欢迎的终端Git工具之一
- 总Star超过63k
- 让Git操作像玩游戏一样轻松
