# cc-switch

## 项目名称
cc-switch

## 作者或组织名
farion1231

## 项目类别
开发工具

## 核心痛点
在不同AI编码工具之间切换需要手动编辑配置文件，管理MCP和Skills很麻烦。

## 主要功能
AI编码工具一体化管理器，支持Claude Code、Codex、Gemini CLI、OpenCode和OpenClaw五种工具。提供50多个预设提供商（AWS Bedrock、NVIDIA NIM、社区中转服务等），一键导入。统一管理MCP服务器和Skills，一个面板管理四个应用，双向同步配置。系统托盘快速切换。支持云同步（Dropbox、OneDrive、iCloud、WebDAV）。本地代理支持热切换、自动故障转移、熔断器。使用统计和成本跟踪。Deep Link导入支持。

## 项目亮点
- 支持5种AI编码工具
- 50+预设提供商
- 统一MCP和Skills管理
- 系统托盘快速切换
- 云同步支持
- 本地代理和故障转移
- 使用统计和成本跟踪
- 原生桌面应用（Tauri 2）

## 安装或部署教程

### 教程适用对象与环境范围
本教程适用于想要使用CC Switch管理AI编码工具的用户。

### 前置条件检查

1. **检查操作系统**
   - 执行命令：无
   - 通过标准：Windows 10+、macOS 12+或主流Linux发行版

### 主路径：Windows安装

**步骤1：下载安装包**
- 目标：获取安装包
- 位置：浏览器
- 操作：访问GitHub Releases页面下载MSI安装包或便携版
- 成功标志：文件下载完成
- 失败处理：检查网络连接 [海外网络环境配置 →](/网络环境)

**步骤2：安装应用**
- 目标：安装CC Switch
- 位置：本地
- 操作：运行MSI安装包，按提示完成安装
- 成功标志：应用安装成功
- 失败处理：检查系统权限

**步骤3：启动应用**
- 目标：运行CC Switch
- 位置：桌面或开始菜单
- 操作：打开CC Switch应用
- 成功标志：应用窗口打开
- 失败处理：检查系统兼容性

### 备选路径1：macOS安装

**步骤1：使用Homebrew安装（推荐）**
- 目标：通过Homebrew安装
- 位置：终端
- 命令：`brew tap farion1231/ccswitch && brew install --cask cc-switch`
- 成功标志：安装成功提示
- 失败处理：检查Homebrew是否正确安装

**步骤2：或手动下载**
- 目标：下载macOS版本
- 位置：浏览器
- 操作：访问GitHub Releases下载macOS版本
- 成功标志：文件下载完成
- 失败处理：检查网络连接

### 备选路径2：Linux安装

**步骤1：下载对应格式**
- 目标：获取Linux安装包
- 位置：浏览器
- 操作：访问GitHub Releases下载deb、rpm、AppImage或flatpak
- 成功标志：文件下载完成
- 失败处理：检查网络连接

**步骤2：安装**
- 目标：安装应用
- 位置：终端
- 命令：根据格式使用对应命令安装
- 成功标志：安装成功
- 失败处理：检查依赖

### 安装成功验证
- 应用正常启动
- 可以添加提供商
- 可以切换不同的AI工具配置

### 常见错误排查

1. **macOS提示"无法验证开发者"**
   - 可能原因：应用未签名
   - 解决步骤：系统设置 → 隐私与安全 → 仍要打开

2. **无法删除当前活跃提供商**
   - 可能原因：系统需要保持一个活跃配置
   - 解决步骤：先切换到其他提供商再删除

3. **配置同步失败**
   - 可能原因：云服务连接问题
   - 解决步骤：检查云服务账号和网络连接
