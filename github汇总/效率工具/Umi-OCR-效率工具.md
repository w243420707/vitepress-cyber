# 项目名称
Umi-OCR

# 作者或组织名
hiroi-sora

# 项目类别
效率工具

# 核心痛点
需要文字识别但不想依赖网络服务，需要批量处理图片文字识别

# 主要功能
- 免费开源的离线OCR软件
- 支持截图OCR、批量OCR、PDF识别
- 支持二维码、公式识别
- 支持忽略区域、批量处理、多种语言界面
- 支持多种图片格式和保存格式

# 项目亮点
- 完全免费，代码开源
- 离线运行，不需要网络
- 解压即用，使用方便
- 批量处理无数量上限
