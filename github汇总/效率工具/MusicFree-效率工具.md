# 项目名称
MusicFree

# 作者或组织名
maotoumao

# 项目类别
效率工具

# 核心痛点
音乐播放器广告多，可定制性差，音源单一，缺乏纯净的音乐体验。

# 主要功能
插件化、定制化、无广告的免费音乐播放器，支持通过插件搜索和播放来自多个音源的音乐。

# 项目亮点
1. 插件化、定制化、无广告
2. 免费音乐播放器
3. 支持通过插件搜索和播放来自多个音源的音乐
4. 具有高度的可定制性和扩展性
5. 采用AGPL-3.0许可证
6. 无广告打扰
7. GitHub上已获得超过6.4k星标
