## 概述

AI 开发工具帮助开发者更高效地开发 AI 应用。本分类汇集了各类优质的开源 AI 开发工具，涵盖模型训练、微调、部署等，帮助您提升 AI 开发效率。

## 工具列表

### Unsloth

- **项目名称**: Unsloth
- **作者**: Unsloth AI
- **核心功能**: 让大语言模型微调和推理速度提升 2 倍，内存减少 50%
- **主要亮点**:
  - 支持超多主流模型（Llama 3、Phi 3、Mistral、Gemma、Qwen、Grok-1 等）
  - 不需要改代码，安装简单
  - 手动优化的反向传播和自动混合精度训练
  - 聪明的 LoRA 和 QLoRA 实现
  - 不损失精度，输出结果完全一样
  - RTX 3060、GTX 1080TI 这种老显卡都能跑
  - 支持 OpenAI API 格式的推理接口
  - 免费 Colab 笔记本可用
  - 最受欢迎的 LLM 训练优化工具之一
- **详细介绍**: [Unsloth 完整文档](/github汇总/AI开发工具/Unsloth-AI开发工具)
- **GitHub 地址**: https://github.com/unslothai/unsloth
