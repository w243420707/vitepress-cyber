# 项目名称
OpenViking

# 作者或组织名
字节跳动火山引擎（Volcengine）

# 项目类别
AI开发工具

# 核心痛点
AI处理长期任务时，背景资料会越来越多，传统做法是每次让AI干活都重新读一遍所有资料，效率低下。

# 主要功能
字节跳动火山引擎开源的专为AI Agents设计的上下文数据库，创新性地引入文件系统范式管理Agent上下文。

# 项目亮点
1. 字节跳动火山引擎开源
2. 专为AI Agents设计的上下文数据库
3. 创新性地引入文件系统范式管理Agent上下文
4. 解决AI处理长期任务时上下文管理问题
5. 刚开源不到一个月，GitHub已飙到8.3k星
