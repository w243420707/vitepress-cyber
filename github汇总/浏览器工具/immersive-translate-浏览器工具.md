# 项目名称
immersive-translate

# 作者或组织名
immersive-translate（中国独立开发者Owen主导）

# 项目类别
浏览器工具

# 核心痛点
外文网页阅读不便，翻译体验差，缺乏沉浸式的双语对照翻译功能，隐私保护不足。

# 主要功能
沉浸式双语网页翻译扩展，支持实时翻译网页、PDF排版还原、本地AI翻译保护隐私。

# 项目亮点
1. 沉浸式双语网页翻译扩展
2. 支持实时翻译网页
3. 支持PDF排版还原
4. 支持本地AI翻译保护隐私
5. 最近新增了BabelDoc功能
6. 可以直接调用各大顶尖大模型
7. 由中国独立开发者Owen主导的个人项目
