# CondFoleyGen

## 作者或组织名
Yuexi Du, Ziyang Chen, Justin Salamon, Bryan Russell, Andrew Owens（密歇根大学、耶鲁大学、Adobe研究院）

## 项目类别
AI音频生成

## 核心痛点
给视频配上合适的音效很麻烦，需要专业拟音师或大量人工音效库

## 主要功能
- 通过拟音类比从视频条件生成音频
- 官方PyTorch实现
- 支持Greatest Hits和Countix-AV数据集
- 提供预训练模型
- 支持训练和推理

## 项目亮点
- 来自密歇根大学、耶鲁大学和Adobe研究院的联合研究
- CVPR收录的研究成果
- 能根据视频内容自动生成匹配的音效
- 提供完整的训练和推理代码
- 支持预训练模型下载
