# 🏗️ AI框架

## 概述

AI 框架是构建 AI 应用的基础。本分类汇集了各类优质的开源 AI 框架，帮助您更高效地开发 AI 应用。

## 工具列表

### LangChain

- **项目名称**: LangChain
- **作者**: LangChain AI
- **核心功能**: 用于构建智能体和 LLM 驱动应用的框架
- **主要亮点**:
  - 帮助链化互操作组件和第三方集成
  - 提供模型、嵌入、向量存储等的标准接口
  - 支持实时数据增强
  - 支持模型互操作
  - 完整的生态系统，包括 LangGraph、LangSmith 等工具
  - 模块化、基于组件的架构，支持快速原型设计和迭代
  - 生产级功能，内置监控、评估和调试支持
  - 活跃的社区和生态系统
- **详细介绍**: [LangChain 完整文档](/github汇总/AI框架/LangChain-AI框架)
- **GitHub 地址**: https://github.com/langchain-ai/langchain

### AutoGen

- **项目名称**: AutoGen
- **作者**: AutoGen
- **核心功能**: 多 Agent 框架
- **主要亮点**:
  - 多 Agent 框架
  - 功能强大
  - 开源
- **详细介绍**: [AutoGen 完整文档](/github汇总/AI框架/AutoGen-AI框架)
- **GitHub 地址**: https://github.com/AutoGen/AutoGen
